# PC Pulse
## Инструкция
1. Скачать файл [PC Pulse.exe](https://drive.google.com/file/d/1CPMpkcynEwN7gzcRjWGI-nwC8aXDF8sd/view?usp=sharing) на ПК, запустить и дать все запрашиваеые разрешения.
2. Скачать файл [PC Pulse.apk](https://drive.google.com/file/d/1yEn6NZc63FclPJRQEXwNdFlJy7SRBLVE/view?usp=sharing) на смартфон и установить приложение.
3. В приложении на смартфоне нажать "+" и в диалоговом окне ввести IP показанный в окне приложения на ПК.
